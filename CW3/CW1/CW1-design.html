ES6 FEATURES IN JAVASCRIPT

CLASS : I have used class in Program Page as embedded script. I have used it from line 151 to 191."The PromotionManager" class manages the program page's promotional information. Its constructor has "promotions," which contains the three promotional messages we need to display, "element," which picks the HTML element to display these messages, and "intervalId," which is a tracking method for the rotation interval.

"getRandomPromo" selects messages at random, "displayRandomPromo" changes the webpage's assigned element with a randomly selected promotion, and "startRotation" cycles through "promotions" at the specified interval (3 seconds) using "setInterval". "stopRotation" halts the rotation, allowing a message to be displayed. The "displayInitialPromo" method ensures that the message is displayed instantly and that the next rotations are triggered after the 3-second wait.

SET :I have used set in apply.js page which is an external script. I have used it from line 170 to 179 and 243.To store the major picked, this set was utilized instead of an array. The main motivation for this was to avoid duplicates. When an array was first utilized, duplicates were placed into the array, causing the "Total Numbers of Major Applied" to be larger than it should have been. As a result, a set with no duplicate values was used.

CONST :I have used it in Visit Page as an embedded script. I have used it in line 89 to 92 and 123.All the variables declared here are for selectors so, they will not need to be changed which will be convenient as “constant” variables cannot be reassigned.

LET :I have used it in Visit page as an embedded script. I have used it in line 147 to 150.Let has been used inside an event listener function. As these variables are only going to be used inside the function or block, having a block scope (which let has) allows us to only have those variables be usable inside the function.
